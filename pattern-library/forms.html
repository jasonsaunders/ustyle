Fields Github: _fields.scss

This is the basic form field, .us-field wrapper for any single input or select. It sets a few layout options by default (margin) and styles any labels within. You can create a fake label style with .us-faux-label

States

  • .us-field-required - Any required form field has this class to show an asterisk beside the label
  • .us-field-blocked - Blocked label styling
  • .us-field-inline - Inline label styling

Example

.us-field-required
.us-field-blocked
.us-field-inline

               <div class="us-field">
                 <label for='name'>Name</label>
                 <input class='us-form-input' id='name' name='name' placeholder='Please enter your name'>
               </div>

Input group Github: _input-group.scss

Inputs groups allow you to add "boxes" to the left or right of an input. These boxes usually contain a visual icon to represent the usage of the input

Selects Github: _select.scss

The select is built with inline background-image SVGs for Webkit/Blink browsers that allow for customisation on the hover/normal colours.

Firefox allows for .us-form-select-wrap to be added on top of the select to replicate styling, whilst other browser fallback to native implementation.

$select-base-color, $select-hover-color and $select-active-color allow for different colouring of the border/dropdown.

States

Example


               <span class='us-form-select-wrap'>
                 <select class='us-form-select'>
                   <option>A select menu</option>
                   <option>With lots of options</option>
                   <option>And a few more</option>
                 </select>
               </span>

Toggle Github: _toggle.scss

This is a form toggle used for single select options. Best usage is for 2 - 3 options that are short answers.

IE8 support is included with radioToggle.js which sets a .checked class on the radio button when clicked.

States

Example


             <div class="us-field us-field-blocked">
               <label>Gender</label>
               <div class="us-toggle">
                 <label class="us-toggle__label" for="gender_male">
                   <input class='us-form-input' name='gender' type='radio' id="gender_male" />
                   <span class="us-toggle__btn">
                     Male
                   </span>
                 </label>
                 <label class="us-toggle__label" for="gender_female">
                   <input class='us-form-input' name='gender' type='radio' id="gender_female" />
                   <span class="us-toggle__btn">
                     Female
                   </span>
                 </label>
               </div>  
             </div>
                

               var radioToggle = new RadioToggle({
                 $target: $(".us-toggle")
               })

Form validation Github: _validation.scss

Validation message and input styling provides a consistent look and feel to uStyle forms.

States

  • .us-validation--bordered - use this for any singular field validation or validation on backgrounds
  • .us-validation--success - Success styling for correct forms
